## Điểm khác biệt so với bản gốc
- ✅ Cô đọng và tổ chức lại từ 861 dòng gốc, giữ full methodology
- ✅ Thêm 3 ví dụ thực tế: Agent Sourcing XNK, Agent Trading Signal, Agent Content TikTok

- ✅ Vietnamese-first triggers, mô tả, và SOUL.md mẫu tiếng Việt

## Cải tiến cụ thể
1. **Agent Brief YAML** có sẵn mẫu cho 4 loại agent Comarai
2. **Memory System** tối ưu cho context trading (daily log format cho trade journal)
3. **Multi-Agent Team** thiết kế cho mô hình Lean của Comarai (1 founder + AI agents)
4. **Safety Guardrails** thêm rules cho trading bot và XNK data
5. **Heartbeat Strategy** phù hợp lịch trading (London open 14:00 VN, NY open 19:30 VN)
